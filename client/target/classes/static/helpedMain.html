Ciao, ti trovi nella sezione introduttiva

Per richiedere il supporto di un tecnico attraverso l'applicazione, svolgi i seguenti passi:

  1. Registrati con un username, selezionando la sezione "registrati" del menù a sinistra. Con tale username riconoscibile all'interno del sistema
  2. Ora puoi sfruttare le funzionalità dell'app: portati sulla sezione "videochiamata" per eseguire la tua prima videochiamata. Per videochiamare il tuo interlocutore basterà inserire il suo username quando richiesto.
  3. Le chiamate che effettui sono registrate automaticamente. Per riprodurle basterà premere "Replay" nella schermata della videochiamata
  4. Per condividire file con il tuo interlocutore portati nella sezione "Condividi file" del menù a sinistra
  5. Per iniziare una chat con il tuo interlocutore portati nella sezione "chat" del menù a sinistra
